# intermediate.int_product_collections

商品×コレクションのブリッジ (名称付与)。粒度: 1 行 = (コレクション × 商品)。

| カラム | 型 | 説明 |
|---|---|---|
| collection_id | varchar | コレクション ID |
| collection_title / collection_handle | varchar | コレクション名 / ハンドル |
| product_id | varchar | 商品 ID |
| product_title / product_type | varchar | 商品名 / タイプ |
